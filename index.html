Project Overview

Concept

The EDSUnity AutoBot is designed to automate WhatsApp messaging with two core features: intelligent auto-replies and automatic message deletion. This serves as an engagement tool while maintaining privacy through ephemeral messaging.

Core Features

Feature Description Status
Auto-Reply Responds to keywords with predefined templates Ready
Auto-Delete Deletes messages after configurable time period Ready
Broadcast Messaging Sends messages to multiple contacts Beta
Activity Logging Tracks messages in Firebase Realtime DB Ready

Workflow

Auto-Reply Triggered
Log to Firebase
Auto-Delete Scheduled

Tech Stack

The solution is built using:

  • whatsapp-web.js: For WhatsApp integration
  • Firebase Admin SDK: For realtime database operations
  • Node Scheduler: For timed auto-delete operations
  • Puppeteer: Headless browser for WhatsApp Web

Code Implementation

Setup and Initialization

bot.js 
const { Client, LocalAuth } = require('whatsapp-web.js');
const admin = require('firebase-admin');

// Initialize Firebase
const serviceAccount = require('./firebase-key.json');
admin.initializeApp({
  credential: admin.credential.cert(serviceAccount),
  databaseURL: "https://edsunity-bot.firebaseio.com"
});

// Initialize WhatsApp Client
const client = new Client({
  authStrategy: new LocalAuth(),
  puppeteer: { 
    headless: true,
    args: ['--no-sandbox', '--disable-setuid-sandbox']
  }
});

// QR Code for Authentication
client.on('qr', (qr) => {
  console.log('QR RECEIVED', qr);
  // In a real app, you would display this QR code to the user
});

client.on('ready', () => {
  console.log('Client is ready!');
  // Bot is ready to receive messages
});

client.initialize();

Auto-Reply and Auto-Delete

bot.js 
client.on('message', async (msg) => {
  // Only process text messages
  if (msg.body && !msg.fromMe) {
    // Respond to 'info' keyword
    if (msg.body.toLowerCase() === 'info') {
      const replyMsg = await msg.reply(
        `Hello! Thank you for your interest. Here's our portfolio: 
https://edsunity.github.io/portfolio-edcy/`
      );
      
      // Log to Firebase
      const messageRef = admin.database().ref('messages').push();
      await messageRef.set({
        from: msg.from,
        to: msg.to,
        message: msg.body,
        reply: replyMsg.body,
        timestamp: new Date().toISOString()
      });
      
      // Schedule auto-delete after 24 hours
      setTimeout(async () => {
        try {
          await replyMsg.delete(true); // Delete for everyone
          console.log(`Message deleted: ${replyMsg.id.timestamp}`);
        } catch (error) {
          console.error('Delete failed:', error);
        }
      }, 86400000); // 24 hours in milliseconds
    }
  }
});

Broadcast Messaging (Beta)

broadcast.js 
async function broadcastMessage(contacts, message) {
  for (const contact of contacts) {
    try {
      const chat = await client.getChatById(contact);
      await chat.sendMessage(message);
      console.log(`Message sent to: ${contact}`);
      
      // Log broadcast
      admin.database().ref('broadcasts').push({
        contact,
        message,
        timestamp: new Date().toISOString()
      });
    } catch (error) {
      console.error(`Failed to send to ${contact}:`, error);
    }
    // Delay to avoid rate limiting
    await new Promise(resolve => setTimeout(resolve, 2000));
  }
}

Deployment Guide

Prerequisites

  • Node.js v16+
  • Firebase project with Realtime Database
  • WhatsApp account for the bot
  • Server with internet access

Installation Steps

Terminal Commands 
# Clone the repository
git clone https://github.com/edsunity/autobot.git
cd autobot

# Install dependencies
npm install

# Set up Firebase credentials
# Download your service account key from Firebase Console
# Save as firebase-key.json in project root

# Start the bot
node bot.js

# Scan the QR code with your WhatsApp account
# The bot is now active!

Environment Configuration

Create a .env file with the following variables:

.env 
# Auto-delete time in hours
AUTO_DELETE_HOURS=24

# Keywords that trigger auto-reply
KEYWORDS=info,help,portfolio

# Broadcast interval in minutes
BROADCAST_INTERVAL=60

# Admin phone numbers (comma separated)
ADMINS=1234567890,0987654321

Ethical & Legal Compliance

Important Notice: This bot is intended for educational and demonstration purposes. Using unofficial WhatsApp automation solutions may violate WhatsApp's Terms of Service. We strongly recommend using the official WhatsApp Business API for production systems.

Best Practices

  • Always obtain explicit consent before adding users to broadcast lists
  • Implement rate limiting to avoid being flagged as spam
  • Provide clear opt-out instructions in your messages
  • Respect user privacy - don't store personal data unnecessarily
  • Comply with local regulations regarding automated messaging

WhatsApp Business API

For production use, we recommend migrating to the official WhatsApp Business API which provides:

  • Compliance with WhatsApp policies
  • Higher message limits
  • Verified business profiles
  • Official support channels
  • Enhanced security features
