WhatsApp AutoBot

Dokumentasi Mini

Panduan ringkas untuk fitur auto-reply & auto-delete dalam format buku mini yang elegan.

Pengantar

Apa itu WhatsApp AutoBot?

Sistem otomatisasi untuk mengelola komunikasi WhatsApp dengan fitur utama auto-reply dan auto-delete.

Fitur Utama

  • Auto-Reply - Balas pesan otomatis
  • Auto-Delete - Hapus setelah periode
  • Broadcast Pesan
  • Log Aktivitas

Teknologi

  • Node.js
  • whatsapp-web.js
  • Firebase

Implementasi

Setup Awal

Inisialisasi client WhatsApp:

const { Client } = require('whatsapp-web.js');

const client = new Client({
  authStrategy: new LocalAuth(),
  puppeteer: {
    headless: true
  }
});

client.initialize();

Auto-Reply

Contoh implementasi:

client.on('message', (msg) => {
  if (msg.body === 'info') {
    msg.reply('Hello! Portfolio: https://edsunity.github.io');
  }
});

Auto-Delete

Implementasi

Hapus pesan setelah 24 jam:

client.on('message', async (msg) => {
  if (msg.body === 'info') {
    const reply = await msg.reply('...');
    setTimeout(() => {
      reply.delete(true);
    }, 86400000); // 24 jam
  }
});

Konfigurasi

Atur waktu di file .env:

# Durasi penghapusan (jam)
AUTO_DELETE_HOURS=24
